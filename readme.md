# Bracket for Smart Orbiter 3 Direct Drive Extruder to Voron Tap

This repository contains 3D print design files for a bracket that attaches a Smart Orbiter 3 direct drive extruder to a Voron Tap toolhead.

## Files

- `oriber3-tap-bracke-second-attempt.FCStd`: FreeCAD source file for the bracket design
- `bracket-Body-smart-orbiter3-to-voron-tap.stl`: STL file ready for 3D printing
- `beta-bracket.jpg`: Preview image of the bracket

## Preview

![Bracket Preview](beta-bracket.jpg)

## Usage

1. Download `bracket-Body-smart-orbiter3-to-voron-tap.stl` for 3D printing
2. Print with appropriate settings for structural parts (recommended: PETG or ABS)
3. Open `oriber3-tap-bracke-second-attempt.FCStd` in FreeCAD if you need to modify the design
